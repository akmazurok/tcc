npm i nodemon

#### Instalar o dotenv
npm install dotenv

#### Clonar repositório
git clone https://github.com/Achinitz/TCC.git

#### Entrar na pasta do projeto front end Web
cd back

#### instalar dependências
npm install

#### executar o projeto
npm start run:dev

#### Executando o Projeto
O sistema está sendo executado em localhost:3000
